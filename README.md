# Ebon Blade — RPG Prototype

**2D-игра на Python + Pygame** с двумя концовками: победой и поражением.  
Игрок управляет рыцарем, сражается с боссом в реальном времени.  
Проект выполнен в рамках учебного экзамена.

---

## Геймплей

- **Цель:** убить босса двумя ударами, избегая его снарядов (ShadowSplash).
- **Игрок:** 100 HP, атака пробелом (50 урона).
- **Босс:** 100 HP, автоматически стреляет каждые 3 секунды.
- **Интерфейс:** healthbar игрока (три состояния), healthbar босса (два состояния).
- **Сохранение:** через меню паузы в JSON-файл.

### Управление

| Клавиша | Действие |
|---------|----------|
| WASD / Стрелки | Движение |
| Пробел | Атака |
| ESC | Пауза / Пропуск сцены / Возврат в меню |

### Концовки

- **Поражение** (игрок умирает): чёрный экран → руки босса → голова → сердце → разбитое сердце → меню.
- **Победа** (босс убит): вид от первого лица — рука рыцаря с мечом замахивается, голова босса раскалывается в 4 стадии → меню.
